## Installation

First clone this github repository, then `cd` into the repository.

Then run in the terminal :

```
python -m venv .
```

**On Ubuntu/Mac (other than M1) :**

```
source bin/activate
```

**Windows :**

```
.\Scripts\activate
```

or if using Git Bash :

```
. Scripts/activate
```

Then run

```
pip install --upgrade pip

python -m pip install -r requirements.txt
```

and finally :

```
python app.py
```
